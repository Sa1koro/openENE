## 两步搞定

### 1️⃣ 装 Tailscale

去 [tailscale.com/download/windows](https://tailscale.com/download/windows) 下载安装，然后用你的账号登录：

```powershell
tailscale up
```

在你手机/笔记本上也装好 Tailscale 并登录同一账号。这样所有设备就都在同一个 **tailnet**（虚拟局域网）里了。

### 2️⃣ 配置 Gateway

```json5
// ~/.openclaw/openclaw.json
{
  gateway: {
    bind: "loopback",          // 保持不变，Tailscale Serve 会代理
    tailscale: {
      mode: "serve",           // ☆ 关键：开启 Tailscale Serve
    },
    auth: {
      allowTailscale: true,    // ☆ 允许 Tailscale 身份验证（默认就 true）
      // ... 你原有的 auth 配置保持不变
    },
  },
}
```

改完后重启 Gateway：

```powershell
openclaw restart
```

### 效果

只要你的手机/笔记本连上了同一个 tailnet，打开浏览器访问：
```
https://<机器名>.<tailnet名>.ts.net:18789
```

就能直接连回 Gateway 的 Control UI，不用暴露公网端口，不用搞 DDNS，流量全部走 WireGuard 加密。

## 进阶选项

| 模式 | 访问范围 | 说明 |
|------|---------|------|
| `"serve"` | 仅 tailnet 内设备 | 最安全，默认推荐 |
| `"funnel"` | **公网也能访问** | 暴露到互联网，会自动绑定 `bind: "tailnet"` 并需要 auth |

如果想从浏览器拿到完整 HTTPS（而不是 `http://...ts.net:18789`），可以搭配 `tailscale funnel` 模式或者自己配反向代理。
